:::info Ziele
- Entwerfen einer Funktionshierarchie für das NumTrip-Game

- Implementation des NumTrip-Games als konsolenbasiertes Programm
- ⭐️ Erweiterung des konsolenbasierten NumTrip-Games mit Pygame
:::
